Photosynth REST API Demo

This application demonstrates how to use the Photosynth REST API in conjunction with Microsoft Account for authentication to create a desktop application for uploading images to create Technical Preview Photosynths. You can use this code as a basis to create your own. For example, you could provide creative and beautiful new ways for presenting a Photosynth user's library, or make a mobile app for capturing and uploading images.

The OpenPhotosynth project is provided as a Portable Class Library, so you can use it across your Windows Store, Windows Phone, Silverlight and, of course, .NET desktop applications.

All of the source code is released under the MIT License; feel free to use, modify and distribute this code as you see fit.

See the official REST API documentation for more detailed information, and use the forum to reach out if you have any questions or issues.
